
# Portfolio

This is a personal portfolio website built with Next.js and Tailwind CSS. It showcases projects, skills, volunteering, and contact information in a modern, responsive design.

## Features

- **Hero Section**: Eye-catching introduction with avatar and branding.
- **About Me**: Brief bio and background.
- **Projects**: Highlights of selected works and case studies.
- **Designs**: Showcases design portfolio or featured visuals.
- **Volunteering**: Details about volunteering experience.
- **Contact**: Contact form or details for reaching out.
- **Responsive Navbar**: Smooth navigation with avatar and resume download.
- **Scroll Animations**: Engaging scroll-based reveals and transitions.
- **Tailwind CSS**: Utility-first styling for rapid UI development.

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- pnpm (or npm/yarn)

### Installation

1. Clone the repository:
	```sh
	git clone https://github.com/Bhagi2001/Portfolio.git
	cd Portfolio/portfolio
	```
2. Install dependencies:
	```sh
	pnpm install
	# or
	npm install
	# or
	yarn install
	```
3. Run the development server:
	```sh
	pnpm dev
	# or
	npm run dev
	# or
	yarn dev
	```
4. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

- `app/` — Main application code (pages, components, styles)
- `public/` — Static assets (images, avatar, etc.)
- `tailwind.config.ts` — Tailwind CSS configuration
- `next.config.js` — Next.js configuration
- `package.json` — Project metadata and scripts

## Customization

- Update content in `app/components/` for About, Projects, etc.
- Replace images in `public/` as needed.
- Adjust theme colors in `tailwind.config.ts`.

## Deployment

You can deploy this project to Vercel, Netlify, or any platform supporting Next.js.
